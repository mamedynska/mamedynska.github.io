Jeu d'échec

Réalisé en binôme, en Java.


Lors de ma deuxième année de DUT, j'ai eu un projet de Java avec un sujet libre. Mon binôme et moi avons décidé d'essayer de recréer un jeu d'échec en Java. Nous avons utiliser comme interface graphique JavaFX. Nous avons commencé par la conception de notre programme avec des diagrammes de classes. Nous avons beaucoup réfléchis à la façon dont implémenter notre échiquier afin de pouvoir créer un jeu de dame à partir de la même base. Pour cela nous avons utiliser les patrons de conceptions que nous avions vu en cours.
A la fin de notre projet, nous n'avons pas pu implémenter l'échec et mat par manque de temps. Cependant nous pouvions jouer une vraie partie et pour gagner il suffisait de manger le roi adverse.

  • Date: Decembre 2018

Projet de Réalité Virtuelle

Réalisé en binôme, en C#


Dans le cours de réalité virtuelle de ma deuxième année de DUT, nous avons réalisé avec mon binôme, un projet sur le Leap Motion. Le Leap Motion est un outil qui permet de capturer les mouvements des mains. Il sert de base à retranscrire ses mouvements dans un monde VR. Dans le cadre de ce projet nous l'avons utiliser afin de créer un jeu de tir.
Ce jeu a été réalisé en C#.

  • Date: Février 2019
